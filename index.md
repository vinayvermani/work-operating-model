# Personal Work System

Simple model of working on the Job. This suppose to help in managing work, learning and reduce anxiety related to work, deadlines and communication.

## 1. TODO List (Daily today/tomorrow) - Required

- make daily todo list in the begining of the day,
- make daily closing todo list of the next day,
- todo can contain work items, personal items and learning items etc.
- no stress should be taken for the format of the todo and structure
- idea is to think and write down the work that I would like to achieve today and the next day

## 2. Question List/Bank (Running) - Based on Thinking

produce list of questions

- These questions could be related to things that needs to addressed and answered to move ahead
- These questions help to produce the Todo Items based on the questions, also block other todos becauase they need these answers
- For question, how some thing works? needs to be done to finish todo item that needs this knowledge first

## 3. Artifacts (Notes, mindmaps, knowledge articles etc) - Based on Thinking/Learning

Produce daily artifacts(notes, mindmaps, diagrams, lists, knowledge arcticle)

- Ugly is fine: rough notes, messy flow diagrams, half-formed lists; enough that future-you can understand and reuse it
- This is bring confidence, long term memory and solutioning things
- can also referred during finish a todo item

## 4. Crutial Communication Log (Running) - Based on Thinking/Situations

Keep list of critical communication questions/concerns

- what are confusions and concerns
- Critical questions and worry
- responses, actions and sample responses
- Whats in control? Whats out of control?
- Whats is mising? who can get it? What I have done?
- reference to todo items that can be in control like providing support, seeting meetings and askign questions

## 5. Weekly Close - Required

- Help with relaxing
- plan next week
- address next weeks concern
- Plan monday
- plan tough conversation for next week
- plan items that needs attention during weekend
- What cannot be done in weeekend and why? and only refer moday todo
